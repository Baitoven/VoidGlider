# Setup

* Setup your .env with a correct `DISCORD_TOKEN` and a valid `FFMPEG` .exe location
* Probably need to run 
```shell
python -m pip install -U discord.py
python -m pip install yt_dlp asyncio dotenv
```
* Run & enjoy
```shell
python VoidGlider.py
```
